### Languages and Tools
- Vue Js 3
- Tailwindcss
- Google Font Poppins
- FontAwesome

- Vite Js
- Acode code editor

### Clone This Repo
- <code>git clone https://github.com/bagussatoto/Doctalk.git </code>
- <code>cd Doctalk</code>
- <code>npm install</code>
- <code>npm run dev</code>
